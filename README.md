# basi-a.github.io
# 下载依赖
```bash
cd hexo
npm install
```
***
# 草稿
## 新建草稿
```bash
cd hexo
./new-draft.sh "文章标题"
```
## 预览草稿
```bash
cd hexo
./hexo-s-draft.sh
```
## 发布草稿
```bash
cd hexo
./hexo-p.sh "文章标题"
```
